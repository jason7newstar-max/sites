## 工具栈 / 技术标准

```
Spline:        3D scenes (free tier OK 起步,Pro 需要时升 $9/mo)
Three.js:      复杂自定义 3D 用
GSAP:          滚动动画(free 普通 license,商用要 $99/年)
Lenis / Locomotive Scroll: 平滑滚动
Framer Motion: React 组件动画
Next.js:       默认主框架(可 static export 也可 SSR)
Tailwind:      快速 CSS
shadcn/ui:     基础组件
Vercel:        默认部署
Sanity / Notion CMS: 客户能自己改内容(选配)
```

## 关键风险

1. **客户改稿无止境** — 写在合同:每 tier 固定 2-3 轮修改,超出 +$500/轮
2. **3D 性能问题**:Spline 重场景在弱设备卡 → 默认 mobile 简化版 + auto-fallback
3. **客户不懂 3D 价值**:你做的 hero 客户觉得"动来动去花哨" → 提案时给 A/B 对比图,清楚说明每秒滞留时间提升
4. **价格被砍** — 给的 $2K 起就是底,SPARK 不能低于 $1.5K(那不如不做)
5. **同质化**:Spline 用得多了 hero 看起来都一样 → 第 6 个月起每个项目至少有 1 个原创 3D asset
